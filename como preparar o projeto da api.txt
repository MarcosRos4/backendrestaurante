Passo a Passo pra fazer a api funcionar meus bacanas

1 - inicia um projeto em uma pasta qualquer, de preferiencia chamada "backend" já no seu projeto do react
comando: yarn init -y

2 - instala o typescript
comando: yarn add typescript -D

3 - instala o express e sua tipagem
comando1: yarn add express
comando2: yarn add @types/express -D

4 - inicia o typescript no projeto
comando: yarn tsc -init

5 - baixa mais biblioteca
comando: yarn add express-async-errors

6 - continua escrevendo comando ai
comando1: yarn add cors
comando2: yarn add @types/cors -D

7 - baixa o postgreSQL (aplicativo PgAdmin4 e a database padrão)
https://www.postgresql.org
Usuário: postgresql
Senha: admin
Porta: 5432
database: rendezvous

8 - instala o Prisma agora
comando: yarn add prisma
comando: yarn add @prisma/client


9 - depois de tudo isso arrasta os arquivos pra pasta back end

10 - pra iniciar o server da api
comando: yarn run dev
